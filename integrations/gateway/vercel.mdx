```json
{
  "mcpServers": {
```

</Accordion>

<Note> This integration is for the **Respan gateway**. </Note>

## What is AI SDK?

The [AI SDK](https://ai-sdk.dev/) (by Vercel) is a TypeScript toolkit for building AI-powered applications with Next.js, React, and other frameworks. Route LLM calls through the Respan gateway to access 250+ models with a single API key.

## Quickstart

### Step 1: Install packages

```bash
npm install ai @ai-sdk/openai
```

### Step 2: Set environment variables

```bash
export RESPAN_API_KEY="YOUR_RESPAN_API_KEY"
```

### Step 3: Point the provider to the Respan gateway

<Tabs>
<Tab title="OpenAI">
```typescript
import { createOpenAI } from '@ai-sdk/openai';
import { generateText } from 'ai';

const openai = createOpenAI({
  apiKey: process.env.RESPAN_API_KEY!,
  baseURL: 'https://api.respan.ai/api',
});

const result = await generateText({
  model: openai('gpt-4o-mini'),
  prompt: 'Tell me a joke',
});
```
</Tab>
<Tab title="Anthropic">
