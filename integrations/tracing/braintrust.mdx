## What is Braintrust?

[Braintrust](https://www.braintrust.dev/) is an AI evaluation and observability platform. The Respan exporter registers as Braintrust's background logger to mirror experiment and logging data to Respan.

## Setup

<Steps>
<Step title="Install packages">

```bash
pip install braintrust respan-exporter-braintrust openai
```

</Step>

<Step title="Set environment variables">

```bash
export RESPAN_API_KEY="YOUR_RESPAN_API_KEY"
export BRAINTRUST_API_KEY="YOUR_BRAINTRUST_API_KEY"
export OPENAI_API_KEY="YOUR_OPENAI_API_KEY"
```

</Step>

<Step title="Initialize and run">

```python
import braintrust
from openai import OpenAI
from respan_exporter_braintrust import RespanBraintrustExporter

client = OpenAI()

with RespanBraintrustExporter():
    @braintrust.traced
    def generate_joke():
        response = client.chat.completions.create(
            model="gpt-4o-mini",
            messages=[{"role": "user", "content": "Tell me a joke about AI"}],
        )
        return response.choices[0].message.content

    @braintrust.traced
    def joke_pipeline():
        return generate_joke()

    result = joke_pipeline()
    print(result)
```

</Step>

<Step title="View your trace">

Open the [Traces page](https://platform.respan.ai/platform/traces) to see your Braintrust traces in Respan.

</Step>
</Steps>

## Configuration

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `api_key` | `str` | `RESPAN_API_KEY` env var | Respan API key. |
| `base_url` | `str \| None` | `None` | API base URL. |

See the [Braintrust Exporter SDK reference](/sdks/python/exporters/braintrust) for the full API.
